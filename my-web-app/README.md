# مشروع Libre Network

شبكة بلوكتشين لا مركزية بآلية إثبات النشاط (Proof of Activity)، مع عملتها الخاصة **Libre Coin (LBRC)**.

---

## مميزات المشروع

- إثبات النشاط (PoA) مع سك تدريجي للعملة.
- قاعدة بيانات محدثة تحتوي على:
  - قائمة المستخدمين العامة.
  - قائمة المعدنين (تُحدث كل 20,000 تحديث).
  - مجمع المعاملات.
- أنواع معاملات متعددة:
  - تحويل عملات بين المستخدمين.
  - زيادة العمر التعدينى.
  - إنشاء حساب جديد.
  - تغيير ملكية الحساب.
  - دخول مجمع المعدنين.
- آلية تعدين تجمع المعاملات وتنتج تحديثات مشفرة (مع هاش وتوقيع مجمع).
- حذف الحسابات غير النشطة بعد انتهاء صلاحيتها.
- حرق الرسوم وتوزيع مكافآت التعدين.
- واجهة مستخدم حديثة وسهلة الاستخدام (React).
- API Backend مبني على Express وMongoDB.

## تشغيل المشروع

### المتطلبات

- Node.js v16+
- MongoDB مثبت ويعمل على localhost أو URI خاص بك
- متصفح حديث لتشغيل الواجهة

### الخطوات

1. **تشغيل Backend:**

```bash
cd libre-network/backend
npm install
node index.js



تشغيل Frontend:


cd libre-network/frontend
npm install
npm run dev



افتح المتصفح على العنوان الذي يظهر في الطرفية (عادة http://localhost:3000)



استخدام التطبيق


وافق على شروط الاستخدام.


استخدم صفحة المحفظة لإدارة أرصدتك وإرسال المعاملات.


استخدم صفحة التعدين لإنشاء تحديثات الشبكة والحصول على مكافآت.


راجع حالة الشبكة والمعاملات في الواجهة.

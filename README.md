مشروع Libre Network
نظرة عامة

Libre Network هي شبكة بلوكتشين لا مركزية مصممة باستخدام لغة Python، تحتوي على عملتها الرقمية الخاصة Libre Coin (LBRC).
تعتمد الشبكة على تحديث حالة قاعدة البيانات بشكل تسلسلي، حيث يمثل كل تحديث تغييرًا في حالة الشبكة ويتم تأمينه من خلال آلية توافق إثبات النشاط (Proof of Activity).

الهدف هو توفير نظام آمن، قابل للتوسع، ولا مركزي لإدارة الحسابات والمعاملات والمعدنين، مع الحفاظ على سجل كامل لجميع التحديثات من خلال سلسلة هاشات متصلة.

Merkle Tree لكل قائمة

لكل قائمة رئيسية: المستخدمين، المعاملات، المعدنين، مجمع المعدنين

Leaf = hash لكل عنصر (بيانات الحساب، المعاملة، أو المعدن)

Root Hash يمثل الحالة الحالية للقائمة بالكامل

يمكن التحقق من أي عنصر باستخدام Merkle Proof فقط → التحقق سريع جدًا O(log n)
